# 🚀 KSGI 지부 운영 자동화 파트너 (KSGI Branch Partner)

지부 운영 실무의 패러다임을 바꿉니다. 수기로 작성하고 일일이 계산하던 번거로운 행정 업무를 자동화하여, 실무자가 '사람'에 더 집중할 수 있도록 돕는 실무형 관리 솔루션입니다.

---

## ✨ 핵심 비전 (Core Vision)

1. **최소의 입력 (Minimum Input)**:
   - 사진 한 장으로 명단이 입력되고, 기존 데이터가 자동으로 연결됩니다.
2. **즉시 사용 가능한 출력 (Ready-to-use Output)**:
   - 버튼 클릭 한 번으로 인쇄용 회의 안건지와 통계 보고서가 생성됩니다.
3. **흐름 중심의 관리 (Flow-centric)**:
   - 월별 활동의 흐름과 변화가 한눈에 보이는 직관적인 대시보드를 제공합니다.
4. **철저한 로컬 보안 (Local Privacy)**:
   - 모든 데이터는 개인용 컴퓨터(브라우저)에만 저장되어 외부 유출 우려가 없습니다.

---

## 🛠 주요 기능 (Key Features)

### 📊 스마트 대시보드

- 실시간 참석자 추이 및 확대 달성률 시각화
- 이번 달 핵심 과제와 누락 알림 자동 노출

### 📝 실무 최적화 입력

- 좌담회, 부원회, 청년스쿨 등 활동별 맞춤형 폼
- **[준비 중]** OCR 기술을 활용한 명단 사진 텍스트 자동 인식

### 🖨 인쇄 및 결과물 생성

- **1초 보고서**: 예정된 일정과 실적을 조합한 완벽한 회의 안건지 출력
- **다양한 포맷**: PDF 저장, 엑셀 내보내기, 프린트 최적화 레이아웃 제공

### 🤝 멘토링 매칭

- 멘토-멘티 매칭 현황 및 1:1 활동(근행회 등) 누적 기록

---

## 🏗 프로젝트 구조 (Structure)

- **Frontend**: React (Vite) + Tailwind CSS + Lucide Icons
- **Database**: Dexie.js (IndexedDB) - 로컬 기반 고성능 데이터베이스
- **Architecture**: 기능 중심의 컴포넌트 분리 및 문서 중심 설계

---

## 📅 로드맵 (Roadmap)

- **Phase 1 (MVP)**: 기본 대시보드 및 입력 폼 구축, 인쇄 레이아웃 세팅
- **Phase 2 (Automation)**: OCR 기능 연동, 고급 통계 분석 대시보드
- **Phase 3 (Optimization)**: 백업/복구 시스템, UI/UX 고도화
