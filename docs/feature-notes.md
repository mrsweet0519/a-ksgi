# KSGI 운영 대시보드 - 주요 기능 명세 (Feature Notes)

이 문서는 KSGI 지부 운영 실무를 자동화하기 위한 핵심 기능들을 정의합니다.

## 1. 데이터 입력 계층 (Input Layer)

### OCR 자동 입력 (최우선)

- 실무자가 손으로 쓴 명단이나 회의록 사진을 업로드하면 텍스트를 인식합니다.
- 인식된 텍스트를 기반으로 `Member` 또는 `Meeting` 데이터를 자동 생성 제안합니다.
- **목표**: 타이핑 시간 80% 단축

### 지능형 폼 (Intelligent Forms)

- 이전 데이터 기반 자동 완성 (예: 참석자 명단 불러오기)
- 필수 항목 최소화 및 직관적인 UI

## 2. 관리 및 분석 계층 (Management Layer)

### 좌담회 및 일정 관리

- 월별 달력 뷰와 리스트 뷰 제공
- 목표 대비 달성률(참석자 수) 자동 계산 및 시각화

### 멘토링 매칭 시스템

- 멘토와 멘티의 연결 상태 관리
- 1:1 근행회 및 활동 기록 누적

## 3. 출력 및 공유 계층 (Output Layer) - **실무 핵심**

### 자동 보고서 생성기

- **운영회의 안건지**: 예정된 일정과 지난 활동 요약을 바탕으로 1클릭 생성
- **월간 통합 보고서**: 통계 차트와 주요 활동 기록을 포함한 공식 문서 양식

### 출력 최적화 (Print/PDF/Excel)

- **그대로 출력**: 인쇄 시 불필요한 UI(버튼, 사이드바) 자동 제거
- **PDF 저장**: 공식 보관용 PDF 파일 생성
- **엑셀 내보내기**: 원본 데이터 백업 및 추가 편집용 데이터 제공

## 4. 보안 및 성능 (Non-functional)

- **로컬 보안**: 브라우저 내 IndexedDB를 사용해 외부 유출 없이 내 컴퓨터에만 저장
- **속도**: React Query를 통한 빠른 데이터 동기화 및 렌더링
